# 2桁掛け算練習アプリ

2桁の整数（11〜99）同士の掛け算を繰り返し練習するためのWindows専用アプリケーションです。
問題と正解を音声で読み上げる機能があり、計算練習を効率的に行うことができます。
また、間違えた問題や難しかった問題を保存し、後で復習する機能も備えています。

## 機能概要

* **ランダム出題**: 11から99までの2桁の整数の掛け算をランダムに出題します。
* **音声読み上げ**: WindowsのSAPI機能を使用し、問題と正解を音声で読み上げます。
* **リトライ機能**: 苦手な問題をリストに保存し、その問題だけを重点的に練習するモードがあります。

## 必要要件

* **OS**: Windows (音声読み上げに `win32com.client` (SAPI) を使用しているため)
* **Python**: 3.13 以上
* **依存ライブラリ**:
    * `pyside6`
    * `pywin32`
    * `pyyaml`

## インストール方法

1. このリポジトリをクローンまたはダウンロードします。
2. 必要なライブラリをインストールします。

```bash
pip install pyside6 pywin32 pyyaml
```

## 使い方

### 1. 通常練習モード（ランダム出題）

以下のコマンドを実行してアプリケーションを起動します。

```bash
python calc.py
```

**操作方法:**

* **`N` キー**:
    * **出題時**: 新しい問題が生成され、音声で読み上げられます（例:「11かける22」）。
    * **回答時**: 続けて押すと、正解が音声で読み上げられます。
    * これを繰り返すことで、次々と問題を解くことができます。
* **`M` キー**:
    * 現在出題されている問題を「リトライリスト（`retry.txt`）」に保存します。苦手な問題を後で復習したい場合に使用します。
* **`Q` キー**:
    * アプリケーションを終了します。

### 2. リトライモード（復習）

`calc.py` で `M` キーを押して保存した問題を練習するモードです。
※ このモードを実行するには、ディレクトリ内に `retry.txt` が存在する必要があります。

```bash
python calc_from_database.py
```

**操作方法:**

* **`N` キー**: 保存された問題リストからランダムに出題されます。操作手順は通常モードと同じです。
* **`Q` キー**: アプリケーションを終了します。

### 3. リトライリストの削除

保存した問題リスト（`retry.txt`）を削除したい場合は、以下のコマンドを実行します。

```bash
python remove_database.py
```

## 注意事項

* 本アプリケーションは音声が出ますので、音量にご注意ください。
* Windows以外の環境（Mac/Linux）では、音声読み上げライブラリ（`pywin32`）が対応していないため動作しません。
