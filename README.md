# 2桁かけ算練習アプリ (Multiplication Practice)

このアプリケーションは、2桁同士のかけ算（11×11 〜 99×99）を音声読み上げ機能を使って練習するためのWindows専用ツールです。
間違えた問題や苦手な問題を保存し、後で「Retry」モードで重点的に復習することができます。

## 主な機能

- **Practiceモード**: ランダムに出題される2桁のかけ算を練習します。
- **Retryモード**: 保存した苦手な問題を復習します。
- **音声読み上げ**: Windows標準の音声合成エンジン（SAPI）を使用し、問題と答えを読み上げます。

## 必要要件

- **OS**: Windows (SAPIを使用するため)
- **Python**: 3.13以上
- **パッケージマネージャ**: [uv](https://github.com/astral-sh/uv) (推奨)

## インストール方法

このプロジェクトはパッケージ管理に `uv` を使用しています。

1. リポジトリをクローンします。
2. 以下のコマンドで依存関係をインストールします。

```bash
uv sync
```

## 使い方

以下のコマンドでアプリケーションを起動します。

```bash
uv run main.py
```

### 操作方法

キーボードショートカットを使って操作します。

| キー | 動作 | モード |
| :--- | :--- | :--- |
| **n** | 次の問題を出題 / 答えを聞く | 全モード |
| **m** | 現在の問題を「Retryリスト」に保存 | Practiceモードのみ |
| **q** | アプリケーションを終了 | 全モード |

### 各モードの詳細

#### 1. Practice タブ (練習)
- アプリ起動時のデフォルトモードです。
- **n** キーを押すと、「〇〇 かける 〇〇」と問題が読み上げられます。
- もう一度 **n** キーを押すと、答えが読み上げられます。
- 答えを聞いて「間違えた」「難しかった」と感じた場合は、**m** キーを押してリストに保存してください。

#### 2. Retry タブ (復習)
- Practiceモードで **m** キーを使って保存した問題のみが出題されます。
- 保存された問題がランダムな順番で出題されます。
- **n** キーで問題と答えを交互に確認できます。

### データの管理

- **データベースのクリア**: メニューバーの `Database` -> `Clear Database` を選択すると、保存されたRetryリストが全て消去されます。
- 保存データは `retry.txt` というファイルに記録されます。
